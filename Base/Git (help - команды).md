# &nbsp;Git (help - команды)

* git help -a
* git help -g                    показывают доступные подкоманды и концептуальные руководства.  
* git help <command>
* git help <concept>           для чтения о конкретной подкоманде или концепции.  
* git help git                    для обзора системы.

# Основные команды Git для работы с репозиторием:
* git config --global user.name "Ваше Имя"                                    Устанавливает \*\*ваше имя\*\* для всех коммитов
* git config --global user.email "ваш@email.com"                            Устанавливает \*\*ваш email\*\* для всех коммитов - Связывает коммиты с вашим аккаунтом (на GitHub и др.)
* git config --list -                                                                               Показывает \*\*все текущие настройки\*\* Git  - Проверяет, что настройки применились
* git backfill                                                                                          Загрузить отсутствующие объекты в частичном клоне 
* git init                                                                                                Создать новый репозиторий
* git clone <url>                                                                                     Клонировать удаленный репозиторий

# Удаление файлов из git и из репозитория

\# 1. Удаляем ВСЕ отслеживаемые файлы ИЗ ИНДЕКСА (но не с диска)

\# Это нужно, чтобы Git "забыл" о них перед коммитом

git rm -r --cached .

\# 2. Удаляем ВСЕ файлы и папки в текущей директории (кроме .git)

\# rm -r \* удаляет всё, что видно в папке (все файлы проекта)

rm -r \*

\# 3. Добавляем в индекс факт удаления файлов

\# git add -u добавляет изменения в уже отслеживаемых файлах (в т.ч. удалённые)

git add -u

\# 4. Делаем коммит: теперь история знает, что файлов больше нет

git commit -m "Очистка: все файлы удалены"

\# 5. Отправляем изменения на GitHub, перезаписывая историю

git push origin main --force

\# удалить файлы из удаленного репозитория

git push origin main --force
