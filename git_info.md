# Команды git

## git status
### состояние репозитория

## git add file.py
### добавлять в индекс файл file.py

## git add .
### добавить в индекс все файлы в текущем каталоге

## git commit -m "commit for commit"
### создать соммит с комментарием

## git push
### залить изменения с локального репозитория на github

## git pull
### скачать изменения с github в локальный репозиторий

## git remote
### управление связью между текущим и удаленным репозиторием

## git log --oneline
### история коммитов
